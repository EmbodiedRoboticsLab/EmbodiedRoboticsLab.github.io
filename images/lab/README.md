# RICG Laboratory Media

All laboratory-facing website media is organized below `images/lab/`.

```text
images/lab/
├── hero/           # Home-page primary platform photograph
├── projects/       # Featured Projects carousel images
├── people/         # Faculty and student portraits
├── news/           # Home Latest News and /news/ images
└── publications/   # Publication and patent thumbnails
```

Keep the filenames referenced by the HTML files unchanged. Image paths use root-relative URLs (for example, `/images/lab/news/whole-body-mpc.jpg`).
